Lists in HTML

There are two primary types of lists in HTML. The unordered list (<ul>), which is used for collections where sequence does not matter, and ordered list (<ol>) Where sequence does matter.

Both of these lists use the element list item (<li>) for each item in the list.

Tables

Tables allow you to organize data in rows & columns. Semantically, "header" rows should be included in the <thead> and the main section of the table should be in a <tbody> element.

Notice that <th> elements automatically bold the content. These should be used for header cells. The <caption> element in a table gives a caption to the table which is displayed above the table by default. Each row is denoted by the <tr> element, with <td> and <th> elements residing within each row.

One of the harder aspects to master in HTML Tables is the rowspan, which causes the cell to span vertically over rows, and the colspan, which causes the cell to span horizontally over columns.
